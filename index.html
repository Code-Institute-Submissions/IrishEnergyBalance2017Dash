How to use the Dashboard

  • The dashboard can be used on mobile, but is best explored on desktop.
  • Hover the mouse over a data group to reveal additional information on a data group. (Only available on desktop)
  • Click on a data group to filter the data displayed on the charts.
    i.e. Click the 'Oil' data group on the 'Consumption by Fuel' will filter for 'Oil' data across all charts, while clicking 'Transport' data group on the 'Consumption by Consumer' chart will only filter across 'Final Energy Consumption' charts.
  • Clicking a filtered data group again to remove the group from the filter.
  • Narrow or widen your filter range by selecting or removing a number of data groups.
  • Reset the charts data by clicking the 'Reset Charts' option from the menu.

About the Charts

Source of Data

The data source for these graphs is a report called Energy in Ireland 2018 which was prepared by SEAI, and published in December 2018.

Chart Development

The data was downloaded as .xlsx, converted to .json and furtherly reorganised using a Python script.

The interactive graphs were made using a javascript charting library called dc.js, which was built to work natively with crossfilter and leverages d3.js library.

Bootstrap framework was utilised for responsive page developement.

The code and development write up can be found on GitHub.
